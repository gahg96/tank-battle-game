# 钢铁洪流：东线战场 (Iron Clash: Eastern Front) - 游戏说明书

欢迎来到《钢铁洪流：东线战场》！这是一款快节奏的 2D 坦克对战游戏，你将指挥一支装甲小队，在广阔的东线战场上与无穷无尽的敌军展开殊死搏斗。

## 🎮 游戏目标
指挥你的坦克小队（共 3 辆），消灭源源不断的敌方坦克。击杀敌人可以获得分数并提升关卡等级。随着关卡提升，敌人会变得更强、更多，但你的小队也会获得补给和修复。

## 🕹️ 操作指南

### 移动与视角
*   **移动**：`W` `A` `S` `D` 或 `方向键`
*   **瞄准**：`鼠标移动` 控制炮塔方向
*   **切换视角**：`C` (在 **俯视跟随** 和 **驾驶舱视角** 之间切换)
*   **暂停游戏**：`ESC`

### 武器系统
*   **主炮射击**：`空格键 (SPACE)`
    *   标准反坦克炮，射程远，伤害中等。
*   **同轴机枪**：按住 `V`
    *   射速极快，适合持续压制，但单发伤害较低。
*   **霰弹轰击**：`Q`
    *   一次发射 3 枚炮弹，近距离爆发极高，适合贴脸肉搏。
*   **燃烧瓶**：`F`
    *   向鼠标指向区域投掷燃烧瓶，落地后形成一片火海，对经过的敌人造成持续伤害。
*   **布设地雷**：`E`
    *   在坦克后方布设一枚反坦克地雷。敌人踩中后会受到毁灭性打击（100点伤害）。
*   **空中支援 (大招)**：`R`
    *   当左上角显示 **ULTIMATE READY** 时可用。呼叫地毯式轰炸，瞬间消灭全屏敌人！(冷却时间 30 秒)

### 战术指挥
*   **切换坦克**：`TAB`
    *   你控制一支由 3 辆坦克组成的小队。按 TAB 键可以在存活的坦克之间切换控制权。
    *   你不控制的坦克会由 AI 接管，自动跟随你并攻击敌人。
*   **重生**：`F1`
    *   如果全队覆没，按 F1 重新开始游戏。

## 🗺️ 界面说明 (HUD)
*   **左上角**：显示当前分数 (Score)、关卡 (Level) 以及大招状态。
*   **右上角**：显示当前控制提示。
*   **右下角 - 战术雷达**：
    *   **绿点**：我方坦克。
    *   **红点**：敌方坦克。
    *   **白框**：当前屏幕视野范围。
*   **坦克上方**：
    *   **绿色血条**：当前生命值。
    *   **绿色光圈**：当前玩家控制的坦克。

## ⚠️ 战场环境
*   **电子围栏**：地图边界设有高压电子围栏（青色发光）。触碰会造成持续伤害，请务必远离！
*   **障碍物**：木箱可以阻挡子弹和移动，利用它们作为掩体。
*   **残骸**：被击毁的坦克会留下残骸，残骸具有物理碰撞体积，可以作为临时掩体。

## 💡 战术建议
1.  **不要孤军深入**：利用你的 AI 队友吸引火力，你负责侧翼输出。
2.  **地雷风筝流**：被多辆敌人追击时，一边倒车一边按 `E` 布雷，效果拔群。
3.  **贴脸爆发**：对付落单的强力敌人，冲上去按 `Q` 霰弹骑脸，瞬间秒杀。
4.  **火海封路**：在狭窄路口投掷燃烧瓶 (`F`)，阻断敌人进攻路线。
5.  **时刻关注雷达**：敌人会从四面八方涌来，雷达能救你的命。
