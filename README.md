# 钢铁洪流：东线战场 (Iron Clash: Eastern Front) - 游戏说明书

欢迎来到《钢铁洪流：东线战场》！这是一款快节奏的 2D 坦克对战游戏，你将指挥一支多样化的装甲小队，在广阔的东线战场上与无穷无尽的敌军展开殊死搏斗。

## 🎮 游戏目标
指挥你的坦克小队（共 3 辆），消灭源源不断的敌方坦克。击杀敌人可以获得分数并提升关卡等级。

## 🕹️ 操作指南

### 基础操作
*   **移动**：`W` `A` `S` `D` 或 `方向键`
*   **瞄准**：`鼠标移动` 控制炮塔方向
*   **射击**：`空格键 (SPACE)`
*   **切换武器**：`SHIFT` (在 主炮 / 轨道炮 / 导弹 之间切换)
*   **切换坦克**：`TAB` (在小队成员之间切换)
*   **切换视角**：`C` (俯视 / 驾驶舱)
*   **帮助/暂停**：`F1` (查看操作说明)
*   **重生**：`ENTER` (全队覆没后)

### 特殊技能
*   **同轴机枪**：按住 `V` (高射速，低伤害)
*   **霰弹轰击**：`Q` (近距离爆发)
*   **燃烧瓶**：`F` (区域持续伤害)
*   **布设地雷**：`E` (后方布雷)
*   **空中支援 (大招)**：`R` (全屏轰炸，需充能)

## 🛡️ 坦克类型 (Tank Classes)
你的小队由以下三种坦克组成，善用它们的特性：

1.  **突击坦克 (Assault)**
    *   **特点**：属性平衡，万金油。
    *   **武器**：标准火炮 (Standard Cannon)。
2.  **狙击坦克 (Sniper)**
    *   **特点**：移动速度快，但装甲薄弱。
    *   **武器**：轨道炮 (Railgun) - 射速极快，伤害极高，适合远程点杀。
3.  **爆破坦克 (Demolisher)**
    *   **特点**：装甲极厚，但移动缓慢。
    *   **武器**：导弹发射器 (Missile) - 造成大范围爆炸伤害，适合清理密集敌人。

## ⚠️ 战场环境
*   **电子围栏**：地图边界设有高压电子围栏。
*   **障碍物**：利用木箱和残骸作为掩体。
